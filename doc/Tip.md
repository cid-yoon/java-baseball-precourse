### 메일 템플릿

```text
제목 : [$이름] 프리코스 미션 제출합니다.
내용 : 
다음 두 개의 정보를 반드시 포함해 메일을 보낸다.

* 교육 과정 신청시 email 주소: 
* Pull Request URL: 

미션을 진행하면서 느끼고, 배운점, 많은 시간을 투자한 부분 등도 포함하면 더 좋을 것 같아요.
```
